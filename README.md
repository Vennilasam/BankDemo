Bank Management System

A Java-based banking application that demonstrates authentication, authorization, and customer management.
Users can securely log in and perform operations based on their assigned roles.

Features 

Authentication :
Secure user login system
Validates credentials before granting access

Authorization : 
Role-based access control
Different permissions for different user roles (eg. Admin, User)

Customer Management (restricted by role):
Add new customers
View existing customers
Delete customers
